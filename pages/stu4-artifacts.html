6.0.0 STU4 Artifacts defined as part of the CDC Opioid Prescribing Guideline Implementation Guide

NOTE: this implementation guide uses STU3 resources. Therefore, the following STU4 artifacts are only available for download or as raw xml in your browser.

NOTE: as of 3/18/2018, validation frameworks for STU4 resources don't seem to be current. This has caused issues when trying to POST these artifacts to an STU4 FHIR server.

The following STU4 artifacts represent the behavior for implementing CDC Opioid Prescribing Guidelines:
